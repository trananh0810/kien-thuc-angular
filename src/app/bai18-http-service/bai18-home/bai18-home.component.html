HttpClient

HttpClient dùng để gọi dữ liệu từ api.

Ví dụ:

ID: {{userItem.id}}

{{userItem.name}}

Cách sử dụng:

- Bước 1: Import HttpClientModule vào app.module.ts

- Bước 2: Tạo service bằng cmd 'ng g s service_name'.
+ Import HttpClient, Observable.
+ Trong contructor, khai báo 1 biến private http.
+ Tạo method có kiểu dữ liệu trả về là Observable< any[] >
+ Trong method này, sử dụng this.http.get< any[]>('path_api') để lấy dữ liệu từ api về.

- Bước 3: Tại nơi muốn lấy dữ liệu, import service và gọi đến method của service.

Lưu ý: Đây chỉ là hướng dẫn của phương thức get. Các phương thức khác như post, put, delete có thểm xem hướng dẫn thực hiện tương tự tại Http Angular
